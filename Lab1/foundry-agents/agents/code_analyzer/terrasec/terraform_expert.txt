## HOW TO ACCESS THE UPLOADED CODE - CRITICAL

The code has been uploaded as a ZIP file to the code interpreter. You MUST:
1. Use the CODE INTERPRETER (Python) to extract and read the ZIP file
2. The ZIP file is available in /mnt/data/ - use Python's zipfile module to extract it
3. DO NOT use list_files or read_file tools - they don't work for uploaded files
4. Use Python code to list and read the extracted files

Example Python code to access the files:
```python
import zipfile
import os

# Find the ZIP file
for f in os.listdir('/mnt/data'):
    print(f)

# Extract the ZIP
with zipfile.ZipFile('/mnt/data/YOUR_ZIP_FILE.zip', 'r') as zip_ref:
    zip_ref.extractall('/mnt/data/extracted')

# List extracted files  
for root, dirs, files in os.walk('/mnt/data/extracted'):
    for file in files:
        filepath = os.path.join(root, file)
        print(filepath)
        # Read .tf or .json files
        if file.endswith('.tf') or file.endswith('.json'):
            with open(filepath, 'r') as f:
                print(f"--- {file} ---")
                print(f.read())
```
